# Changelog

Releases are generated by release-please and published from tagged builds.

## 1.0.2

- Refactor Dockerfile generation internals into grouped build/runtime/supply-chain configuration models while preserving compatibility.
- Strengthen benchmark subsystem reliability with explicit command timeouts, container cleanup, and optional benchmark dependencies.
- Expand plugin architecture to six extension points with reference plugin examples.
- Add Spring AOT and native AOT Dockerfile recipe support.
- Improve docs organization by consolidating benchmark comparison content into benchmark methodology.

## 1.0.1

- Fix release publishing to keep cosign signature/certificate files out of `dist/`.

## 1.0.0

- First stable release of `springdocker`.
- Includes CI hardening, benchmark workflow maturity, and release automation updates.

## 0.1.0

- Initial `springdocker` CLI package.
- Dockerfile generation and benchmark orchestration commands.
- Sample Spring Boot project and benchmark assets.
