# Docker Best Practices Presentation

This directory contains the Reveal.js deck for Java/Spring Dockerfile decision-making and benchmarking.

## Open locally

```bash
cd /path/to/java-spring-docker
python3 -m http.server 8000
```

Open:

- `http://localhost:8000/docs/presentation/index.html` in your browser to view the presentation.

## Files

- `index.html`: main slide deck
- `assets/custom.css`: deck styling

## Notes

- Reveal.js is loaded from CDN (no local npm setup required).
