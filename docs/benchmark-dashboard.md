# Benchmark dashboard

> **NOT IMPLEMENTED YET**
>
> This page is roadmap content and does not describe currently shipped `springdocker` functionality.

`docs/presentation/index.html` is the generated benchmark dashboard for this repository.

## What it shows

- scenario-by-scenario benchmark comparisons
- trend-friendly summary tables
- security and runtime policy context alongside performance results

## Inputs

- `results/raw.csv` from the benchmark runner
- summary JSON generated by `springdocker benchmark analyze`
- baseline JSON used for regression checks

## Trend reporting

See `docs/examples/benchmark-trends.json` for a compact sample of the trend payload shape.

## CI integration

The CI workflow uses the benchmark regression gate to keep trend lines honest when sample results drift.

## Regeneration

The dashboard is intentionally static so it can be versioned with the docs and reviewed like any other artifact.
